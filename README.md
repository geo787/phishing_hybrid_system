# Hybrid Phishing Detection System

A Machine Learning-based phishing detection system developed as part of my Master's dissertation in Cybersecurity.

The project combines machine learning, natural language processing, and cybersecurity techniques to detect phishing attacks and support security analysts through automated threat analysis.

## Features

- Machine Learning-based phishing detection
- Berlekamp–Massey algorithm integration
- REST API
- Automated threat analysis
- Risk-oriented phishing classification
- Medical-domain phishing detection
- Docker containerization
- Automated reporting

## Technologies

- Python
- Scikit-learn
- Docker
- REST API
- Pandas
- NumPy

## Cybersecurity Concepts

- Threat Detection
- Threat Intelligence
- Risk Assessment
- Phishing Analysis
- Secure Software Development
- Data Analysis

## Project Goals

The objective of this project is to improve phishing detection by combining machine learning techniques with cybersecurity principles and automated threat analysis. The system is designed to support analysts by identifying phishing patterns and assisting risk assessment through structured data processing.
